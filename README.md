## Why this exists

Most Claude resources assume you know how to code. This one doesn't.

This library is for the researcher who wants to synthesize 40 papers in an 
afternoon. The teacher who wants to create lesson plans in minutes. The 
professional who wants to turn a meeting full of notes into a clear action plan.

Every prompt here is tested, explained in plain language, and ready to use 
immediately — no API keys, no terminal, no programming knowledge required.

## How to use a prompt

1. Open [claude.ai](https://claude.ai) in your browser
2. Find a prompt from this library
3. Copy it and paste it into Claude
4. Replace the parts in `[brackets]` with your actual information
5. Hit send

That's it. No setup. No accounts beyond Claude itself.
